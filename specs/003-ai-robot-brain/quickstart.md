# Quickstart: The AI-Robot Brain – NVIDIA Isaac™

## Prerequisites
- Basic knowledge of ROS 2
- Understanding of simulation concepts
- Access to NVIDIA Isaac documentation
- Node.js 18+ and npm installed
- Git for version control

## Setup and Installation
1. Clone the repository: `git clone <repository-url>`
2. Navigate to the project directory: `cd <project-name>`
3. Install dependencies: `npm install`
4. Start the Docusaurus development server: `npm start`

## Module Structure
The module consists of three main chapters:
1. **Chapter 1**: Isaac Sim and synthetic data generation
2. **Chapter 2**: Isaac ROS and accelerated perception (VSLAM)
3. **Chapter 3**: Nav2 and humanoid navigation concepts

## Navigation and Usage
- Access the module at: `/docs/modules/003-ai-robot-brain/`
- Start with the module overview: `/docs/modules/003-ai-robot-brain/index.md`
- Progress through chapters sequentially for best learning experience
- Use the summary page to review the complete pipeline
- Test your knowledge with assessment questions

## Getting Started
- Begin with Chapter 1 to understand Isaac Sim fundamentals
- Progress to Chapter 2 to learn about accelerated perception
- Complete with Chapter 3 for navigation concepts
- Each chapter includes practical exercises and assessments
- Review the glossary for key terminology
- Complete the assessment to validate your understanding

## Development Commands
- To start the development server: `npm start`
- To build the static site: `npm run build`
- To serve the built site locally: `npm run serve`
